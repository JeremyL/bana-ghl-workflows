## Key Decision Paths

| Response Type | Action |
|---|---|
| Positive response | Human-led due diligence → Contract → Close |
| No response | Recurring marketing campaigns (optional) |
| Stop / angry response | Remove, DNC, no further contact |
| Contract not signed | Automated reminders → Re-engage or long-term drip |
| Long-term drip, no response | Follow Up Forever (fully automated) |
| Long-term drip, re-engages | Loop back into pipeline |
